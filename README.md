# MounteBank

A simple mockservice project based on MounteBank

# pre-reqs
 * Node.js

# Clone this repo
 ```http
git clone https://github.com/rama873/mountebank.git
```
Install the dependecies.
 ```http
  npm install
```

# To start the server
 ```http
  npm start
```

This repo is used for the blog
